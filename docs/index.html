Install for Edge
Chrome & Brave
Download latest release from GitHub
Download Release
  1. Download the latest release from GitHub Releases and unzip it.
  2. Open chrome://extensions (or edge://extensions).
  3. Enable Developer mode using the toggle in the top-right corner.
  4. Click Load unpacked and select the siteip folder.
  5. The This IP extension is now installed and active!

The extension icon may be hidden in the extensions menu. To pin it:

  1. Click the puzzle-piece icon (🧩) in your browser toolbar.
  2. Find This IP in the list.
  3. Click the pin icon (📌) next to it.
  4. The ⬡ icon will now appear permanently in your toolbar.

For Edge Add-ons, updates apply automatically. To force a refresh on a manually-installed version:

  1. Download the new release from GitHub Releases.
  2. Open chrome://extensions or edge://extensions.
  3. Click the Refresh icon on the This IP card.
